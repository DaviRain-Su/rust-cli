# rust-cli

## USAGE

```bash
Usage: rcli <COMMAND>

Commands:
  csv      Show csv or Convert CSV to other formats
  genpass  Generate a random password
  base64   Base64 encode or decode
  help     Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
